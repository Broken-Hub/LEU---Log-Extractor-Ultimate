# LEU-Log-Extractor-Ultimate
Leu converts your Hytale log.txt into a clean, structured diagnostic report, automatically classifying crashes, errors, warnings, and affected mods.

Leu is an advanced diagnostic tool built specifically for Hytale.

When the game crashes or starts behaving unexpectedly, simply upload your log.txt file.
Leu automatically processes it and generates a new file called:

log-diagnostico.txt

This generated report includes:

✔ Detailed crash classification

✔ Clear separation between errors and warnings

✔ Detection of mod conflicts

✔ Structured, readable formatting

✔ Final mod status summary

At the bottom of the report, Leu analyzes every mod mentioned in the log and assigns a status label:

🔴 Crash Cause

🟠 Critical Error

🟡 Warning

🟢 No Issues Detected

Instead of manually scanning thousands of technical log lines, Leu turns chaotic data into organized, actionable information.
